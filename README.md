# Personal-Intro-Page

Self learning project, for HTML/CSS/JavaScript practice



\## Purpose

This project was created as a self-learning exercise to quickly acquire front-end development skills.



\## Features

\- Clean, centered layout with gradient background

\- Interactive button with alert message

\- Custom styling with hover effects



\##Technologe

\- HTML5

\- CSS3 (Flexbox, gradients, transitions)

\- JavaScript (DOM manipulation, event handling)
